Pertanyaan yang Sering Diajukan

Temukan jawaban untuk pertanyaan umum seputar jasa pembuatan website kami

Berapa biaya pembuatan website?

Biaya pembuatan website bervariasi tergantung paket yang dipilih. Paket Website Starter mulai dari Rp 2.999.000, Paket Bisnis Pro Rp 4.999.000, dan Paket Custom Enterprise sesuai kebutuhan. Semua harga sudah termasuk domain, hosting, dan dukungan teknis.

Apakah ada biaya bulanan atau tersembunyi?

Tidak ada biaya tersembunyi. Setelah pembayaran awal, Anda hanya perlu membayar domain dan hosting tahunan (termasuk dalam paket tahun pertama). Tidak ada biaya bulanan untuk maintenance website.

Berapa lama waktu pengerjaan website?

Waktu pengerjaan tergantung kompleksitas: Paket Starter (7-10 hari kerja), Paket Bisnis Pro (10-15 hari kerja), dan Paket Custom (15-30 hari kerja). Kami akan memberikan timeline detail saat konsultasi awal.

Bagaimana proses pembayarannya?

Pembayaran dilakukan 50% di awal dan 50% setelah website selesai dan Anda puas dengan hasilnya. Kami menerima transfer bank, e-wallet, dan pembayaran online. Invoice akan dikirimkan untuk setiap tahap pembayaran.

Apa itu domain dan hosting, apakah saya perlu membelinya sendiri?

Domain adalah alamat website Anda (contoh: www.bisnisanda.com), sedangkan hosting adalah tempat menyimpan file website. Keduanya sudah termasuk dalam paket kami untuk tahun pertama. Setelah itu, Anda bisa memperpanjang sendiri atau kami bantu.

Apa itu SEO dan apakah sudah termasuk dalam paket?

SEO (Search Engine Optimization) adalah teknik untuk membuat website muncul di halaman pertama Google. Semua paket sudah termasuk optimasi SEO dasar. Paket Bisnis Pro ke atas mendapat optimasi SEO lanjutan untuk hasil maksimal.

Apakah website akan responsif di mobile?

Ya, semua website yang kami buat sudah responsif dan akan tampil sempurna di desktop, tablet, dan smartphone. Ini sangat penting karena mayoritas pengunjung website saat ini menggunakan perangkat mobile.

Apakah website sudah termasuk SSL certificate?

Ya, semua website sudah termasuk SSL certificate gratis. Ini membuat website Anda aman (HTTPS) dan merupakan faktor penting untuk SEO Google. Pengunjung akan merasa lebih aman saat mengunjungi website Anda.

Apakah saya bisa mengelola konten website sendiri?

Ya, untuk Paket Bisnis Pro ke atas, Anda akan mendapat panel admin (CMS) untuk mengupdate konten dengan mudah. Kami juga memberikan pelatihan lengkap sehingga Anda bisa mengelola website sendiri tanpa perlu pengetahuan teknis.

Berapa lama dukungan teknis yang diberikan?

Dukungan teknis diberikan sesuai paket: Paket Starter (1 bulan), Paket Bisnis Pro (3 bulan), dan Paket Custom (6 bulan). Untuk masalah darurat, tim kami siap membantu 24/7 melalui WhatsApp.

Apa yang terjadi jika website bermasalah setelah diluncurkan?

Kami memberikan garansi 100% untuk semua website. Jika ada masalah teknis, kami akan memperbaikinya gratis selama masa garansi. Untuk masalah darurat, tim support kami siap membantu melalui WhatsApp atau email.

Apakah ada pelatihan yang diberikan?

Ya, semua paket sudah termasuk pelatihan penggunaan website. Kami akan mengajari Anda cara mengupdate konten, mengelola produk, dan menggunakan fitur-fitur website. Pelatihan bisa dilakukan online atau offline sesuai kebutuhan.

Apakah saya akan memiliki website sepenuhnya setelah selesai?

Ya, Anda akan memiliki website sepenuhnya. Semua file website, database, dan kode sumber akan diserahkan kepada Anda. Anda bebas memindahkan website ke hosting lain atau melakukan modifikasi sesuai kebutuhan.

Bagaimana jika saya tidak puas dengan hasilnya?

Kami memberikan garansi revisi unlimited sampai Anda puas dengan hasilnya. Jika setelah revisi berkali-kali Anda masih tidak puas, kami akan mengembalikan uang Anda 100%. Kepuasan klien adalah prioritas utama kami.
